# Monitorización y Data Mining en Ciberseguridad

Detrás de cada infraestructura de TI se generan masivamente flujos de datos por máquinas. Este repositorio contiene el análisis de dichos datos como parte fundamental de la gestión de seguridad en sistemas, desde la detección de amenazas hasta la transmisión de información forense.

## Objetivos del Proyecto

El proyecto se enfoca en el uso de herramientas y métodos adecuados para tratar grandes volúmenes de datos y su correcta visualización:
* **Detección:** Identificar amenazas, fallos y comportamientos anómalos.
* **Inteligencia:** Transmitir de forma inteligible la información forense adquirida.
* **Monitorización:** Seguimiento de Indicadores de Compromiso (IOC) y patrones de adversarios.

## Tecnologías y Metodologías

### Parte 1: Minería de Datos y Data Science
* **Lenguaje R:** Uso de R para estadística, minería de datos y herramientas gráficas.
* **Machine Learning:** Aprendizaje automático y detección de patrones de comportamiento.
* **Experimentación:** Métodos para importar datos, exportar resultados y reproducir experimentos.

### Parte 2: Monitorización y SIEM
* **SIEM:** Técnicas de *Security Information and Event Management*.
* **IOC:** Búsqueda de señales del adversario e Indicadores de Compromiso.
* **Infraestructura:** Monitorización de hardware, software y fuentes de seguridad.

## Bloques Temáticos

- **Registros del sistema:** Peculiaridades de logs en Windows y Linux.
- **Seguridad en redes:** Monitorización de protocolos, puertos y backdoors.
- **Automatización:** Sistemas de detección y correlación de logs heterogéneos.
- **Ciencia de Datos:** Principios de Cloud Computing y Big Data.
- **Minería de Datos con R:** Aplicación específica a la ciberseguridad.

## Estándares y Cumplimiento
El análisis profundiza en la garantía de red y el cumplimiento normativo:
* **Normas ISO**
* **Guías STIC**
* **Sistemas de Control Industrial**
